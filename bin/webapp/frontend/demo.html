Octapus – Demo

This demo page shows a preconfigured scenario that you can run without any setup. Example steps:

  1. Run an Nmap scan on your local /24 network
  2. Automatically enqueue Gobuster + Nikto + SQLmap against any HTTP hosts found
  3. Attempt SSH brute-force via Hydra on any SSH hosts
  4. Collect and display output live in the Logs page

To try it out, go to the Scenario page, click “Load Scenario” → “Demo”, and then “Run Scenario”. All steps will execute in sequence and show live output in Logs.
