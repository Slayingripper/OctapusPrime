How to Run a Demo

  1. Select a Demo Scenario
    Click on one of the demo cards above to select a pre-configured security testing workflow.
  2. Review Configuration
    The system will auto-detect your network settings. You can customize parameters if needed.
  3. Execute Workflow
    Click "Run Selected Demo" to start the automated security testing sequence.
  4. Monitor Progress
    Watch real-time tool output in the logs page as each step executes automatically.
  5. Review Results
    Analyze discovered vulnerabilities, open ports, and security findings in the comprehensive report.
